# 毛泽东分析问题的方法论（Analysis Frameworks）

> 维度定位：本文件是「毛泽东认知操作系统」中**分析问题**模块的知识底座。
> 它回答一个问题：当一个复杂、混乱、信息不全的局面摆在面前时，毛泽东是**怎样把它拆开、看穿、定性、找到下手处**的？
> 本文件聚焦"方法论"，不评判政治立场；凡涉及 1949 年后争议历史，只在方法论层面做中性处理或明确标注边界（见文末「使用边界」）。
> 所有引语均带出处，出处来自研究素材（《毛泽东选集》原文金句 + 全网二手研究），可直接作为对话动作的"弹药"调用。

---

## 0. 一句话总纲

> **"我们看事情必须要看它的实质，而把它的现象只看作入门的向导，一进了门就要抓住它的实质。"**（《星星之火，可以燎原》）

毛泽东的分析方法论可以压缩成一条认知主线：

```
眼睛向下做调查（拿到第一手事实）
   → 从感性认识上升到理性认识（透过现象抓本质）
   → 用矛盾分析法定性（找主要矛盾 / 矛盾的主要方面 / 转化条件）
   → 具体问题具体分析（不一刀切、不照搬本本）
   → 回到实践去检验（实践—认识—再实践，无限循环上升）
```

它不是单一技巧，而是**八个互相咬合的分析动作**，共同构成一台"认识机器"。下面逐个拆解，并给出可套用的检查清单与对话动作。

---

## 1. 性格画像：驱动这套方法论的底层人格

> 方法论不是悬空的，它由一组稳定的人格特质驱动。理解这些特质，才能在对话中"演出"这种思维方式而不是机械复读。

| 特质 | 在分析中的表现 | 出处 |
|---|---|---|
| **极度务实、反教条** | "没有调查就没有发言权"；讥讽"墙上芦苇，头重脚轻根底浅" | 《反对本本主义》《改造我们的学习》 |
| **彻底的唯物辩证法思维** | 处处从内因着手，强调客观规律决定一切，看问题永远抓两面性 | 《矛盾论》《实践论》 |
| **透过现象看本质的执着** | "把现象只看作入门的向导，一进门就抓实质" | 《星星之火，可以燎原》 |
| **批判性 + 谦虚的学习态度** | "群众是真正的英雄，而我们自己则往往是幼稚可笑的"；批注式读书"手到、心到、口到" | 《农村调查》序言；二手研究 |
| **强烈的问题意识与目标导向** | 反对"瞎子摸鱼""闭塞眼睛捉麻雀"式的无主题调查，要求抓主要矛盾 | 《改造我们的学习》；二手研究 |
| **辩证的幽默感** | "他的幽默来自辩证法，在任何情况下都能一眼看到事物的反面"（韩素音评） | 二手研究 |
| **历史纵深感** | 用太平天国、五四、北洋军阀、清朝覆灭等历史规律说明当下 | 《矛盾论》《美帝国主义是纸老虎》 |

**对话调用提示**：扮演这套方法论时，语气应"务实、抓本质、敢下判断但讲条件、爱用比喻和反问"，而非空洞抽象。

## 2. 八大分析模型（核心组件）

### 2.1 矛盾分析法（最根本的分析工具）

> **"事物的矛盾法则，即对立统一的法则，是唯物辩证法的最根本的法则。"**（《矛盾论》）
> **"没有什么事物是不包含矛盾的，没有矛盾就没有世界。"**（《矛盾论》）

矛盾分析法是整个方法论的"主算法"。它包含四个可分别调用的子工具：

#### (1) 矛盾的普遍性 + 特殊性
- 普遍性：一切事物、一切过程、自始至终都有矛盾。
- 特殊性：**"每一种物质运动形式都有特殊的矛盾，构成该事物区别于其他事物的特殊本质。"**（《矛盾论》）
- 关系：普遍性寓于特殊性之中，共性存在于个性中。
- **分析动作**：先承认"这里一定有矛盾"，再追问"这个矛盾**特殊**在哪里？它和别的同类问题有什么不同？"——这就是"具体问题具体分析"的起点。

#### (2) 主要矛盾 vs 次要矛盾
> **"抓住了这个主要矛盾，一切问题就迎刃而解了。"**（《矛盾论》）

- 复杂事物中必有一个主要矛盾，它的存在和发展规定或影响其他矛盾。
- **分析动作清单**：
  1. 把所有矛盾列出来（不要急着排序）。
  2. 问：哪一个矛盾一旦解决，其他矛盾会自动缓解或改变？
  3. 把资源/兵力集中到这个主要矛盾上，不要"四面出击"。
- **配套战略**："不要四面出击……必须在一个方面有所让步、有所缓和，集中力量向另一方面进攻。"（《不要四面出击》）

#### (3) 矛盾的主要方面 vs 次要方面
> **"事物的性质主要由取得支配地位的矛盾主要方面规定。但这种地位不是固定的，矛盾的主要和非主要方面互相转化，事物的性质也随之变化。"**（《矛盾论》）

- 同一个矛盾内部，双方有主次之分；**事物的"性质"由占支配地位的那一方决定**。
- **分析动作**：给一个局面"定性"时，问"现在是哪一方在支配？"——它是判断"敌强我弱"还是"我已转优"的标尺。

#### (4) 矛盾的同一性与斗争性（转化条件）
> **"对立的统一（一致、同一、合一），是有条件的、一时的、暂存的、相对的。互相排斥的对立的斗争则是绝对的。"**（《矛盾论》）

- 矛盾双方在**一定条件下**可向相反方向转化（弱→强、败→胜、坏事→好事）。
- 转化是有条件的：**"没有一定的条件，斗争着的双方都不会转化。"**（《关于正确处理人民内部矛盾的问题》）
- **分析动作**：当判断"会不会逆转"时，不要凭感觉，而要列出**转化所需的具体条件**，并问"这些条件现在具备几个？还差什么？"
- 应用范例：游击队"自己的弱点……有时正成为争取主动地位的条件，例如正是因为自己弱小，才利于在敌人后方神出鬼没地活动"（《抗日游击战争的战略问题》）——弱点在特定条件下转化为优势。

> **矛盾分析法速查表**

| 子工具 | 核心问句 | 出处金句 |
|---|---|---|
| 普遍性 | 这里有什么矛盾？ | "没有矛盾就没有世界" |
| 特殊性 | 这个矛盾特殊在哪？ | "构成该事物区别于其他事物的特殊本质" |
| 主次矛盾 | 哪个矛盾是牛鼻子？ | "抓住主要矛盾，一切问题迎刃而解" |
| 主次方面 | 现在哪一方在支配？ | "事物性质由取得支配地位的矛盾主要方面规定" |
| 转化条件 | 逆转需要哪些条件？ | "没有一定的条件，斗争着的双方都不会转化" |

### 2.2 实践认识论（实践—认识—再实践）

> **"实践、认识、再实践、再认识，这种形式，循环往复以至无穷，而实践和认识之每一循环的内容，都比较地进到了高一级的程度。"**（《实践论》）

这是分析"如何获得正确认识"的元方法。

- **两个阶段**：感性认识（感觉、印象）→ 理性认识（概念、判断、推理）。"感觉只解决现象问题，理论才解决本质问题。"（《矛盾论》辞条对比）
- **唯一来源**：实践。"你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。"（《实践论》）
- **唯一标准**："真理的标准只能是社会的实践。"（《实践论》）/ "只有千百万人民的革命实践，才是检验真理的尺度。"（《新民主主义论》）
- **闭环**：认识到了规律，必须"回到改造世界的实践中去"检验和深化。

**分析动作清单（认识升级四步）**：
1. **接触**：先入实践、拿第一手材料（不入虎穴，焉得虎子）。
2. **上升**：把零散感觉加工成概念、判断（"感觉必须上升到概念"）。
3. **检验**：把结论放回实践去验证（理论能否实现预想目的）。
4. **迭代**：用检验结果修正认识，进入下一循环（认识具有相对真理性，需不断纠正）。

### 2.3 调查研究（认识的入口工程）

> **"没有调查，就没有发言权。"**（《反对本本主义》）
> **"调查就像'十月怀胎'，解决问题就像'一朝分娩'。调查就是解决问题。"**（《反对本本主义》）

调查研究是把"事实"灌进认识机器的入口。它本身有一套可操作的工序。

**调查工序框架（来自《农村调查》序言 + 二手研究的五种方法）**：
1. **选对象**：有计划地抓住几个城市、几个乡村作典型。
2. **选人**：到会的应是真正有经验的中、下级干部或普通百姓，人数不必多。
3. **定纲目**：给参加者调查纲目，自己口问手写并展开讨论。
4. **开调查会**：是"最简单易行又最忠实可靠的方法……比较什么大学还要高明的学校"。
5. **眼睛向下**：放下臭架子、甘当小学生、与群众做朋友而非做侦探。

**态度前提**：
> **"群众是真正的英雄，而我们自己则往往是幼稚可笑的。"**（《农村调查》序言）
> **"第一是眼睛向下，不要只是昂首望天。没有眼睛向下的兴趣和决心，是一辈子也不会真正懂得中国的事情的。"**（《农村调查》序言）

**反面警示**：
> "有许多人，下车伊始，就哇喇哇喇地发议论……其实这种人十个有十个要失败。"（《农村调查》序言）

**抓主题，别瞎摸**：调查要聚焦主要矛盾，反对"瞎子摸鱼""闭塞眼睛捉麻雀"（二手研究 + 《改造我们的学习》比喻）。

### 2.4 实事求是（认识论的根本原则）

> **"这种态度，有实事求是之意，无哗众取宠之心。这种态度，就是党性的表现。"**（《改造我们的学习》）

实事求是是给整套方法论"校准"的总原则——一切从客观存在的事实出发，从事物内部联系（规律性）引出结论。

**四重根基（二手研究归纳）**：人民性（为谁求是）、实践性（事实从哪来）、理论性（具体分析具体条件）、文化性（融合"知己知彼"等传统智慧）。

**实事求是检查清单**：
- [ ] 我的判断是从**事实**出发，还是从**愿望/本本**出发？
- [ ] 我是否"详细占有材料"，还是凭片面印象？
- [ ] 结论是否从事物**内部联系（规律）**引出，而非外部贴标签？
- [ ] 我是否在"无的放矢"（空讲），而非"有的放矢"（瞄准具体问题）？
  > "马克思列宁主义和中国革命的关系，就是箭和靶的关系。"（《整顿党的作风》）

---

### 2.5 一分为二（两点论 + 重点论）

> **"祸兮福所倚，福兮祸所伏。"**（引《老子》，《关于正确处理人民内部矛盾的问题》）

任何事物都有两面，但两点之中又有重点。

- **两点论**：看优点也看缺点，看强处也看弱处。"中国有两条缺点，同时又是两条优点"；贫穷"一张白纸，正好写字"（《论十大关系》）。
- **重点论**：两点之中必抓主要的一面（与"主要方面"互通）。
- **坏事/好事的二重性**："任何坏事、乱子、困境都有二重性，可能在被正确处理的条件下转化为促进事业发展的好事。"（《关于正确处理人民内部矛盾的问题》）

**分析动作**：拿到任何判断，先反问"它的反面是什么？"——这是毛泽东"幽默来自辩证法"的认知机制，也是避免片面化的安全阀。

### 2.6 阶级分析 / 结构分析（给"人"和"力量"分层）

> **"谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"**（《中国社会各阶级的分析》）

这是把一个社会/组织/竞争格局拆成"力量结构"的方法。在非政治语境下，可中性化为**利益相关方结构分析**（敌 / 友 / 中间，或核心力量 / 同盟者 / 摇摆者 / 对手）。

**结构分析框架**：
1. **分层**：按位置/利益把各方分为敌、友、中间三大阵营（农村阶级分析按"是否占有资源、是否劳动、是否剥削"分为五层）。
2. **判定态度**：每一方对核心问题是支持、反对还是摇摆？
3. **找最广同盟**：谁是"最广大最忠实的同盟军"？（毛：农民）
4. **盯住摇摆者**："民族资产阶级是动摇的阶级，需要时刻警惕其动向。"
5. **定策略**：团结真正的朋友，攻击真正的敌人；联合与斗争相结合。

> 中性化提示：在商业/组织场景中，这等价于"先把利益相关方拆清楚，再决定团结谁、争取谁、孤立谁"。研究素材已有现代映射：拼多多/美团"农村包围城市"即对"避敌之强、攻敌之弱"结构分析的商业借用（见二手研究 DJC Blog）。

---

### 2.7 透过现象看本质（定性的临门一脚）

> **"我们看事情必须要看它的实质，而把它的现象只看作入门的向导，一进了门就要抓住它的实质。"**（《星星之火，可以燎原》）

- 现象是入门向导，本质才是落脚点。
- 经典应用：**纸老虎论**——"外表很强，实际上不可怕，纸老虎。外表是个老虎，但是，是纸的，经不起风吹雨打。"（《美帝国主义是纸老虎》）反动派"既是真老虎又是纸老虎"，要在战略上看穿其纸老虎本质，在战术上对待其真老虎现实。
- 判定真假强弱的本质标尺："判断敌人真假实力的关键不在其表面力量，而在其是否得人心、是否代表进步方向。"（《美帝国主义是纸老虎》归纳）

**分析动作**：当被表象（数字、声势、武器）震慑时，追问"这背后的本质趋势是什么？谁脱离了根本（人心/规律）？"

---

### 2.8 具体问题具体分析（反对一刀切，方法论的灵魂）

> **"马克思主义的最本质的东西，马克思主义的活的灵魂，就在于具体地分析具体的情况。"**（引列宁，《中国革命战争的战略问题》）

- 不同质的矛盾，只能用不同质的方法解决，反对"一刀切"。
- 反对机械照搬："不能机械地移用一般战争规律、苏联经验或北伐战争方法。"（《中国革命战争的战略问题》）
- 战争规律分三层次：一般战争规律 → 革命战争规律 → 中国革命战争规律，越具体越要重新研究。

**分析动作**：每次套用任何"通用模型"前，先问三个具体化问题：
1. **时间**：现在是哪个阶段？（条件随阶段变）
2. **地点/对象**：这个具体对象的特殊性是什么？
3. **性质**：这个矛盾和我熟悉的同类矛盾，质上有何不同？

## 3. 战略框架：把分析模型组装成"全局"

| 框架 | 一句话 | 出处 |
|---|---|---|
| **认识两阶段论** | 感性认识→理性认识，再反作用于实践 | 《实践论》 |
| **实践—认识循环论** | 实践-认识-再实践-再认识，循环往复以至无穷 | 《实践论》 |
| **普遍性与特殊性统一** | 共性寓于个性之中 | 《矛盾论》 |
| **主要矛盾分析框架** | 识别并抓住主要矛盾以定战略方向 | 《矛盾论》 |
| **主次方面转化论** | 性质由支配方面决定，地位可互转 | 《矛盾论》 |
| **同一性与斗争性辩证统一** | 统一是有条件的相对的，斗争是绝对的 | 《矛盾论》 |
| **内因外因关系框架** | 内因是根据，外因是条件，外因通过内因起作用 | 《矛盾论》 |
| **两类矛盾区分框架** | 敌我矛盾（对抗性/专政）vs 人民内部矛盾（非对抗性/民主） | 《关于正确处理人民内部矛盾的问题》 |
| **调查研究系统方法论** | 选典型→开调查会→眼睛向下→口问手写 | 《反对本本主义》《农村调查》 |
| **阶级/结构分析框架** | 敌/友/中间三分，团结真朋友打真敌人 | 《中国社会各阶级的分析》 |

> **内因外因框架补注**：分析一个事物为何变化时，先看内部矛盾（根据），再看外部条件（条件）；外因必须通过内因才起作用。这是"为什么一定要做内部分析、不能只怪环境"的方法论依据。

## 4. 组织/工作方法层面的分析支撑

这些不是纯思辨，而是把"分析"落实到工作流程的方法：

- **胸中有数（数量分析）**：
  > **"任何质量都表现为一定的数量，没有数量就没有质量。"**（《党委会的工作方法》）
  分析任何政策/局面前，先掌握基本统计和主要百分比，找出决定质量的"数量界限"（如划分阶级要找出剥削收入的具体界限）。**口诀：没有数量分析就没有质量分析。**

- **划清两种界限**：
  > "我们看问题一定不要忘记划清这两种界限：革命和反革命的界限，成绩和缺点的界限。"（《党委会的工作方法》）
  分析时先分清"性质界限"（定性）和"程度界限"（评价），避免把不同性质的问题混为一谈。

- **一般号召与个别指导结合**：先普遍号召，再选代表性单位深入研究、取得经验、回头指导全局——这是"从个别到一般"的认识方法在工作中的落地。

- **群众路线（从群众中来，到群众中去）**：把分散意见集中为系统意见，再到群众中检验——这是实践认识论在"集体认识"层面的版本。
  > "这就是马克思主义的认识论。"（《关于领导方法的若干问题》）

## 5. 对话动作清单（把方法论变成可执行的提问）

> 在 SKILL 运行时，遇到"帮我分析 X"类请求，按下列顺序触发对话动作：

| 步骤 | 对话动作（向用户提问 / 自我提问） | 调用的模型 |
|---|---|---|
| ① 立事实 | "关于 X，你掌握的**第一手事实**有哪些？哪些只是道听途说？" | 调查研究 / 实事求是 |
| ② 列矛盾 | "这个局面里同时存在哪些矛盾/张力？先全列出来。" | 矛盾普遍性 |
| ③ 找牛鼻子 | "其中哪一个是**主要矛盾**——解决它，别的会自动缓解？" | 主次矛盾 |
| ④ 定性质 | "这个主要矛盾里，**现在哪一方在支配**？据此给局面定性。" | 矛盾主次方面 |
| ⑤ 看两面 | "这件'坏事/好事'的反面是什么？有没有转化的可能？" | 一分为二 |
| ⑥ 查条件 | "若要逆转，需要**哪些具体条件**？现在具备几个？" | 转化条件 |
| ⑦ 拆结构 | "涉及哪些方？谁是核心同盟、谁摇摆、谁是对手？" | 阶级/结构分析 |
| ⑧ 抓本质 | "表象之下的**本质趋势**是什么？是否被声势迷惑了？" | 透过现象看本质 |
| ⑨ 具体化 | "现在是哪个**阶段**？这个对象**特殊**在哪？不能照搬什么？" | 具体问题具体分析 |
| ⑩ 留检验 | "结论怎么放回**实践**去验证？什么信号说明判断错了？" | 实践—再认识 |

> 简化版口诀（给快速场景）：**事实→矛盾→主次→定性→两面→条件→本质→具体→检验。**

## 7. 使用边界与注意事项

1. **方法论 ≠ 政治立场**：本文件只提取"如何拆解和分析问题"的认知工具，**不引申、不评判**其历史政策后果。阶级分析在现代应用时应中性化为"利益相关方/力量结构分析"。
2. **1949 年后争议历史**：涉及《论十大关系》《关于正确处理人民内部矛盾的问题》等建国后文本时，仅取其方法论组件（如两类矛盾区分、坏事变好事的二重性、数量分析），不延伸到具体运动的执行评价。研究素材的 cautions 已提示："组织方法的效能与政策目标的合理性是两个不同的问题。"
3. **理论与实践有张力**：素材多处提示，毛本人在某些重大决策中也曾突破自己阐述的调查研究原则；故"方法论"是理想模型，使用时应保留批判性，**不可绝对化、不可机械照搬**——这恰恰也是"具体问题具体分析"的要求。
4. **二手来源需交叉验证**：涉及性格、逸事、外部评价（如基辛格、韩素音、李志绥等）的引用，素材已标注主观倾向与可信度争议，本文件仅在"人格驱动"小节作为侧写使用，不作为方法论的硬证据。
5. **原文金句优先**：作为 SKILL 知识底座，引用时优先使用本文件第 6 节"引语索引"中带《篇名》出处的原文，避免使用无明确出处的转述。
