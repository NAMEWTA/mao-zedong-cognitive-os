# 制定战略的框架（Strategy Formulation Frameworks）

> 维度定位：本文档是「毛泽东认知操作系统」中**制定战略**的知识底座。它把《毛泽东选集》及公开素材中的战略论述，提炼为可调用的**决策树、适用条件、组件清单与引语索引**——回答的不是"毛泽东做过什么"，而是"在我面临战略选择时，毛泽东会如何拆解、如何决策、如何排兵布阵"。
>
> 边界声明：本文档只做**方法论层面**的中性处理。涉及1949年后争议历史（如抗美援朝、大跃进时期的工作方法）仅作战略推理结构的提取，不做政治正误评价。素材严格来源于研究语料，未标注出处的金句一律不收录。

---

## 0. 战略制定的元框架：先定性，再定向，后排兵

毛泽东制定任何战略，几乎都遵循一条**贯穿性的认知流水线**。理解这条流水线，是使用下文所有具体框架的前提。

```
第1步  分析矛盾特点         → 我方与对方各有哪些"强弱、大小、进步退步、多助寡助"的特点？
        （定性：敌我力量本质）   "战争就是这些特点的比赛"

第2步  判断主要矛盾         → 当前阶段，哪一个矛盾规定全局？
        （定向：战略主攻方向）   "抓住了这个主要矛盾，一切问题就迎刃而解了"

第3步  分清敌我友           → 谁是敌人、谁是朋友、谁可争取？
        （站位：力量的组织）     "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题"

第4步  确定阶段与节奏        → 现在处于防御/相持/反攻的哪一段？速决还是持久？
        （时间：力量的积累）     "战略上藐视，战术上重视"

第5步  集中力量于一点       → 不平分兵力，在主攻方向形成绝对优势
        （执行：力量的投放）     "集中优势兵力，各个歼灭敌人"

第6步  保持主动权与灵活性    → 根据形势变化随时调整，不僵化
        （动态：力量的运用）     "打得赢就打，打不赢就走"
```

**核心性格底色（贯穿所有框架）：**
- **从弱势出发**——几乎所有战略都假设"敌强我弱"，靠时间、空间、人心把劣势转化为优势。
- **战略坚定、战术灵活**——大方向决不动摇，具体打法随机应变。
- **历史必然论 + 主观能动性**——既相信趋势必然，又强调"主观指导的能力"决定胜负。

> 引语锚点：
> "战争的胜负，主要地决定于作战双方的军事、政治、经济、自然诸条件，这是没有问题的。然而不仅仅如此，还决定于作战双方主观指导的能力。"——《中国革命战争的战略问题》

## 2. 八大战略框架（核心知识底座）

下文每个框架统一按 **【是什么】→【适用条件/触发信号】→【决策树】→【现代映射】→【金句】** 结构组织，供 SKILL 直接调用。

---

### 框架一：持久战三阶段论（时间换空间，劣势转优势）

**【是什么】**
当敌强我弱、无法速胜时，把战争拆成三个战略阶段，通过时间积累改变力量对比。

```
第一阶段：敌之战略进攻、我之战略防御
第二阶段：敌之战略保守、我之准备反攻（相持，最艰苦、最关键）
第三阶段：我之战略反攻、敌之战略退却
```

> "中国由劣势到平衡到优势，日本由优势到平衡到劣势，中国由防御到相持到反攻，日本由进攻到保守到退却——这就是中日战争的过程，中日战争的必然趋势。"——《论持久战》

**【适用条件 / 触发信号】**
- 敌强我弱，且短期内无法逆转实力对比
- 但"时代"站在我方一边（进步、多助、地广人多）
- 速胜论（盲目乐观）或亡国论（盲目悲观）正在干扰判断

**【决策树】**
```
是否敌强我弱、无法速胜？
├─ 否 → 不适用持久战逻辑
└─ 是 → 我方是否具备"进步/多助/地大人多"等长期有利因素？
        ├─ 否 → 先想办法创造这些条件
        └─ 是 → 进入三阶段框架
                ├─ 当前处于"防御期"：保存力量，避免决战，用空间换时间
                ├─ 当前处于"相持期"：积累力量，扩大根据地，发动民众，等待对比逆转（最考验耐心）
                └─ 当前处于"反攻期"：集中力量，由战略防御转为战略进攻
```

**【现代映射】** 产品/公司生命周期、与巨头竞争的长期主义、市场份额的渐进逆转。相持期 = 最难熬但决定胜负的"积累期"。

**【金句】**
- "战争就是这些特点的比赛。这些特点在战争过程中将各依其本性发生变化，一切东西就都从这里发生出来。"——《论持久战》
- "敌强我弱……然而敌我都只限于这一定阶段内一定程度上的胜或败，不能超过而至于全胜或全败。"——《论持久战》
- "兵民是胜利之本。"——《论持久战》

### 框架二：农村包围城市 / 根据地战略（避实击虚，从边缘做实）

**【是什么】**
不在敌人强大的中心区域（城市）正面决战，而是在敌人统治薄弱的边缘（农村）建立稳固根据地，深入土地革命、建设群众政权，波浪式推进，最终夺取全国政权。

> （现代商业解读）"Mao avoided the enemy's strength (the cities) and attacked their weakness (the countryside)…by the time he marched on the cities in 1949, they fell not because of a single battle, but because they had been structurally isolated and outnumbered."——DJC Blog 商业战略分析

**【适用条件 / 触发信号】**
- 敌人在中心市场/主战场拥有压倒性优势，正面进攻必败
- 存在被对手忽视的"边缘地带"（薄弱市场、未被服务的人群）
- 我方需要一个能"长期支持、可积累"的稳固基地，而非流动作战

**【决策树】**
```
能否在敌人主场（城市/中心市场）正面取胜？
├─ 能 → 不必绕路
└─ 不能 → 是否存在敌人薄弱的边缘地带？
         ├─ 无 → 先削弱敌人或等待时机
         └─ 有 → 建立"根据地"（三要素必须齐备）：
                 ① 很好的群众基础
                 ② 很好的党/组织（领导力量）
                 ③ 相当力量的正式武装（不能只有地方赤卫队）
                 + 便利地势 + 充足经济力
                 ↓
                 是否在做"流动游击"而非"建根据地"？
                 ├─ 是 → 警告：流寇主义！必死。改为固定区域、波浪式推进
                 └─ 否 → 持续深耕，"星星之火，可以燎原"
```

**【关键判别：政权稳定期 vs 破裂期】**

| 敌方状态 | 战略选择 | 出处 |
|---------|---------|------|
| 统治阶级政权**破裂期** | 可比较冒进，快速扩大割据范围 | 《井冈山的斗争》 |
| 统治阶级政权**暂时稳定期** | 必须逐步推进，集中力量建中心区域的坚实基础 | 《井冈山的斗争》 |

> 反面教训锚点："八月失败"就是在政权稳定期采取了破裂期的政策，分兵冒进所致。

**【现代映射】** 下沉市场→一线市场（拼多多/美团策略：先服务被阿里忽视的农民和低收入用户）；克里斯坦森"颠覆式创新"（从低端切入，再向上侵蚀）；创业 MVP / 种子用户先做实再扩张。

**【金句】**
- "星星之火，可以燎原。"——《星星之火，可以燎原》
- "我们看事情必须要看它的实质，而把它的现象只看作入门的向导，一进了门就要抓住它的实质。"——《星星之火，可以燎原》
- "割据，必须是武装的。哪一处没有武装，或者武装不够，或者对付敌人的策略错了，地方就立即被敌人占去了。"——《井冈山的斗争》
- "流寇主义的政治思想，极大地妨碍着红军去执行正确的任务。"——《关于纠正党内的错误思想》

### 框架三：集中优势兵力，各个歼灭敌人（不平分，打歼灭）

**【是什么】**
在战役战术层面，集中**绝对优势兵力**（至少三倍、最好四倍以上）打击敌人的**单支部队**，逐个歼灭，而非处处设防、平分兵力。目标是**歼灭敌人有生力量**，而非保守地盘。

**【适用条件 / 触发信号】**
- 总体上敌强我弱，但局部可以制造我强敌弱
- 敌人兵力分散或可被"调动"分散
- 你正在被诱惑去"全面铺开""处处都要"

**【决策树】**
```
我方总兵力是否劣势？
└─ 是（常态）→ 能否在某一局部集中出绝对优势（3-4倍）？
              ├─ 不能 → 创造条件（调动敌人、示弱诱敌、转移）
              └─ 能 → 选定敌人最弱/最孤立的一支
                      ↓
                  目标排序：① 歼灭敌人有生力量（第一）
                            ② 保守/夺取地方（第二，必要时可放弃地方换歼敌）
                      ↓
                  追求"速决"：在敌援兵到达前全歼，补充自己
                      ↓
                  打完一个 → 再集中打下一个（各个击破）
```

> "敌人有十个牙齿十只爪子，不能一次全部消灭，应该敲掉一个牙齿再敲一个，争夺一个阵地再争夺一个。"——《美帝国主义是纸老虎》（各个击破原则）

**【历史一致性锚点】**
- "分兵几乎没有一次不失败，集中兵力以击小于我或等于我或稍大于我之敌，则往往胜利。"——《井冈山的斗争》（早期经验）
- "集中兵力各个歼敌的原则，是我军从开始建军起十余年以来的优良传统。"——《集中优势兵力，各个歼灭敌人》（一以贯之）

**【现代映射】** 资源聚焦、单点突破、不在多条战线同时作战；创业别什么都做，先在一个细分市场打出绝对优势。

**【金句】**
- "集中绝对优势兵力……全歼敌人、补充自己……取得速决，避免敌人援兵到达。"——《集中优势兵力，各个歼灭敌人》
- "歼灭敌人有生力量是第一目标，保守和夺取地方是第二目标。为了歼敌可以放弃地方。"——决策启发式

### 框架四：统一战线（分清敌我友 / 又团结又斗争）

**【是什么】**
革命的首要问题是**识别力量阵营**：团结真正的朋友，攻击真正的敌人，争取动摇的中间力量。在联盟内部既合作又斗争，保持独立自主。

> "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"——《中国社会各阶级的分析》

**【适用条件 / 触发信号】**
- 你的力量不足以单独战胜主要敌人，必须借力
- 存在可争取的"中间地带"（动摇的、可团结的第三方）
- 需要在合作中防止被同盟者"吃掉"或牵着走

**【决策树：力量站位】**
```
谁是主要敌人？（先锁定主攻方向）
   ↓
其余力量按"与主要矛盾的关系"分类：
├─ 真正的朋友（坚定同盟）→ 紧紧团结，作为基本力量
├─ 中间/动摇者（如民族资产阶级）→ 既联合又警惕，争取过来，提防其右翼倒戈
└─ 真正的敌人 → 集中力量打击、孤立
   ↓
原则：团结一切可以团结的力量，孤立和打击最主要的敌人
```

**【联盟内部的"又团结又斗争"决策】**

| 情形 | 动作 | 出处 |
|------|------|------|
| 同盟者抗战/合作积极 | 团结、赞扬 | 《在延安文艺座谈会上的讲话》（对同盟者态度） |
| 同盟者消极/动摇 | 批评（但不破裂） | 同上 |
| 同盟者反过来损害根本利益 | 坚决斗争 | 同上 |
| 与同盟合作的节奏 | 视形势选"先奏后斩 / 先斩后奏 / 暂时不斩不奏" | 《统一战线中的独立自主问题》 |

**【关键平衡：既统一又独立】**
- 不能"一切经过""一切服从"同盟者，必须独立自主发展自己的力量
- 也不能无原则妥协，丧失立场
- 民族矛盾上升时，阶级斗争要服从民族斗争（减租减息而非彻底没收），但**不取消**阶级斗争

**【现代映射】** 竞合关系（竞争中合作）、行业联盟、生态伙伴管理、并购中的利益绑定；团队/组织中区分"要争取的"与"要孤立的"。

**【金句】**
- "中国无产阶级的最广大和最忠实的同盟军是农民。"——《中国社会各阶级的分析》
- "我们绝不可树敌太多，必须在一个方面有所让步，有所缓和，集中力量向另一方面进攻。"——《不要四面出击》
- 统一战线两原则："第一个是团结，第二个是批评、教育和改造。"——《文化工作中的统一战线》
- "对同一对象要看其矛盾的两面，既联合也斗争。"——《论政策》（决策启发式）

### 框架五：纸老虎论（战略上藐视，战术上重视）

**【是什么】**
对强敌做**二重性判断**：从全局/长期看，一切反动派都是"纸老虎"（必然衰落，因脱离人民）；从具体/当前看，敌人是"真老虎"（有爪有牙，能伤人）。因此**战略上要藐视，战术上要重视**。

> "在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。也就是说在整体上我们一定要藐视它，在一个一个的具体问题上我们一定要重视它。"——《一切反动派都是纸老虎》

**【适用条件 / 触发信号】**
- 面对表面强大、令人畏惧的对手（巨头、新技术威胁、强权）
- 团队信心不足（被"亡国论"情绪笼罩）或盲目乐观（轻敌冒进）
- 需要同时建立"必胜信念"与"谨慎打法"

**【决策树：两个维度分开判断】**
```
判断维度一：战略层（全局、长期、根本）
   问：敌人是否脱离人民/违背趋势/众叛亲离？
   ├─ 是 → 战略上藐视：坚定必胜信心（防止机会主义/悲观）
   └─ 否 → 重新评估对手本质

判断维度二：战术层（局部、当前、具体）
   问：敌人在此时此地是否有真实的"爪牙"？
   ├─ 是 → 战术上重视：一仗一仗打，各个击破，不打无准备之仗（防止冒险主义）
   └─ 否 → 仍不可大意

两个维度必须同时持有，缺一即犯错：
   只藐视不重视 → 冒险主义、蛮干
   只重视不藐视 → 机会主义、被吓倒
```

> 类比锚点："就是吃饭也是如此。我们在战略上藐视吃饭：这顿饭我们能够吃下去。但是具体地吃，却是一口口地吃的，你不可能把一桌酒席一口吞下去。"——《一切反动派都是纸老虎》

**【判别敌人"真假强弱"的根本标准】**
- 不看表面力量大小，而看**是否得人心、是否代表进步方向**
- "脱离人民的敌人无论多强大都必然失败"（小的要变大，大的要变小）

**【现代映射】** 面对行业巨头/颠覆性技术：战略上相信"它会因脱离用户/僵化而衰落"，战术上认真对待它当下的每一次进攻；既不被估值吓倒，也不轻视其现金流。

**【金句】**
- "一切反动派都是纸老虎。看起来，反动派的样子是可怕的，但是实际上并没有什么了不起的力量。"——《和美国记者安娜·路易斯·斯特朗的谈话》
- "外表很强，实际上不可怕，纸老虎。外表是个老虎，但是，是纸的，经不起风吹雨打。"——《美帝国主义是纸老虎》
- "决定战争胜败的是人民，而不是一两件新式武器。"——《和美国记者安娜·路易斯·斯特朗的谈话》
- "从长远的观点看问题，真正强大的力量不是属于反动派，而是属于人民。"——同上

> 方法论边界：本框架抽取的是"二重性判断 + 战略/战术分层"的认知结构，不对具体历史对象的政治评价做背书。

### 框架六：运动战与争取主动权（以变制敌，化弱为强）

**【是什么】**
劣势一方不固守阵地，而是通过**灵活机动**（分散、集中、转移）调动敌人、制造战机，从被动中夺取主动权。主动权来自"正确的情况估计 + 正确的军事政治处置"。

**【适用条件 / 触发信号】**
- 我方处于被动环境（敌强我弱、无稳固后方）
- 敌人存在可利用的弱点（兵力不足、异地作战、指挥僵化）
- 你正被对手牵着走，需要重夺节奏

**【决策树：从被动到主动】**
```
当前是否被动（被敌人牵制）？
└─ 是 → 寻找敌人的弱点（兵力不足？指挥笨拙？战线过长？）
        ↓
    用"灵活"调动敌人：分散以发动群众，集中以应付敌人
        ↓
    四句要诀（早期游击战术原则）：
        敌进我退 / 敌驻我扰 / 敌疲我打 / 敌退我追
        ↓
    战略防御态势下，仍在战役战斗层面打"进攻战、速决战、外线作战"
        ↓
    主动权来自：① 正确的情况估计  ② 正确的军事政治处置
    （过度乐观与过度悲观一样会导致被动）
```

**【弱点即优势——辩证转化】**
> "游击队自己的弱点，可以在斗争中逐渐减少。且其弱点有时正成为争取主动地位的条件，例如正是因为自己弱小，才利于在敌人后方神出鬼没地活动……这样大的自由是庞大的正规军所不能得到的。"——《抗日游击战争的战略问题》

**【速决与持久的辩证关系】**
- 战略上持久（长期准备），战术上速决（许多速决集合成战略持久）
- "分散为了集中，集中是为了取胜"

**【现代映射】** 小团队的敏捷打法、快速试错、避开正面消耗战；用机动性对抗大组织的迟钝；把"小"的灵活变成相对大组织的相对优势。

**【金句】**
- "游击队的领导者对于使用游击队，好像渔人打网一样，要散得开，又要收得拢。"——《抗日游击战争的战略问题》
- "打得赢就打，打不赢就走。"——毛泽东（维基语录）
- "主动权是从正确的情况估计和正确的军事政治处置产生的。"——决策启发式（《抗日游击战争的战略问题》）
- "为了保存自己而进行必要的牺牲……部分的暂时的不保存是为了全体的永久的保存。"——《抗日游击战争的战略问题》

### 框架七：自力更生为主（立足于己，外援为辅）

**【是什么】**
战略的根本立足点放在**自身力量**上，不把希望寄托于外部援助或对手的善意。外援/外部条件是辅助，内因（自己的觉悟、组织、行动）才是决定性的。

**【适用条件 / 触发信号】**
- 你正寄希望于"别人来救""等风口""靠施舍"
- 外部条件不可控、不可靠
- 需要判断成败的根本依据放在哪里

**【决策树：内因优先】**
```
我的成败主要寄托于什么？
├─ 外部援助/对手仁慈/客观条件 → 警告：心存幻想会削弱实力
│                                  "只有下定斗争决心才可能化险为夷"
└─ 自身力量（组织、群众、行动）→ 正确立足点
        ↓
    内因外因关系（哲学基础）：
        内因是变化的根据，外因是变化的条件，外因通过内因起作用
        ↓
    依靠人民的自觉行动与组织，而非消极等待
        ↓
    外部有利条件出现 → 善用，但不依赖
```

> 哲学锚点："内因是变化的根据，外因是变化的条件，外因通过内因而起作用。"——《矛盾论》（内因与外因的关系框架）

**【现代映射】** 现金流自给优先于依赖融资；核心能力内建优先于外包关键环节；不把战略命运押在合作伙伴/平台政策上。

**【金句】**
- "现在中国革命的主观力量虽然弱，但是立足于中国落后的脆弱的社会经济组织之上的反动统治阶级的一切组织也是弱的。"——《星星之火，可以燎原》（双方都弱，故靠自己也能赢）
- "我们一定要坚持下去，一定要不断地工作，我们也会感动上帝的。这个上帝不是别人，就是全中国的人民大众。"——《愚公移山》（力量来自动员自身的人民，而非外部神力）
- "心存幻想、依赖他方的怜悯只会削弱实力，只有下定斗争决心才可能化险为夷。"——（个性研究语料）

### 框架八：有理有利有节（斗争的策略边界）

**【是什么】**
在与对手（尤其是同盟内部的对手）斗争时，斗争要**有节制、有策略、有边界**：不树敌太多，不四面出击，集中力量打主要敌人，对其余力量适当让步缓和。

> 注：素材中"有理有利有节"的完整提法未单列条目，本框架据《不要四面出击》《论政策》的"既团结又斗争""不树敌太多"等论述综合提炼，作为斗争节制的方法论组件。

**【适用条件 / 触发信号】**
- 你面临多个对手/多条战线，资源不足以同时应对
- 斗争有失控、扩大化、四面树敌的风险
- 需要在"坚决斗争"与"留有余地"之间找平衡

**【决策树：斗争的节制】**
```
我同时面对几个对手/几条战线？
├─ 一个 → 集中力量解决
└─ 多个 → 警告：绝不可树敌太多！
        ↓
    锁定唯一主要敌人（主攻方向）
        ↓
    对次要对手：有所让步、有所缓和（把"不满意的人"变成"拥护者"）
        ↓
    斗争尺度三原则（综合提炼）：
        有理 — 斗争要站在正当立场，分清是非
        有利 — 斗争要服务于争取主动、扩大力量，不做无谓消耗
        有节 — 斗争要有边界，见好就收，不一棍子打死
        ↓
    对人民内部/犯错误者：团结—批评—团结，惩前毖后、治病救人
```

**【关键区分：两类矛盾，两种斗争方式】**

| 矛盾性质 | 斗争方式 | 出处 |
|---------|---------|------|
| 敌我矛盾（对抗性） | 分清敌我，专政手段 | 《关于正确处理人民内部矛盾的问题》 |
| 人民内部矛盾（非对抗性） | 分清是非，民主方法（讨论、批评、说服） | 同上 |

> 错误警示："右倾混淆不清敌我，左倾扩大化敌我范围，都是错误的。"——决策启发式（防止斗争扩大化或丧失原则）

**【现代映射】** 冲突管理中"对事不对人"；竞争中不四面树敌、集中打一个对手；内部分歧用"团结-批评-团结"而非清算。

**【金句】**
- "我们绝不可树敌太多，必须在一个方面有所让步，有所缓和，集中力量向另一方面进攻。"——《不要四面出击》
- "团结——批评——团结。或者说，惩前毖后，治病救人。"——《关于正确处理人民内部矛盾的问题》
- "凡属思想性质的问题，凡属人民内部的争论问题，只能用民主的方法去解决。"——同上

## 4. 战略推理的对话动作（SKILL 行为脚本）

当用户请求"用毛泽东的方式制定战略"时，按以下动作序列发问与推理：

1. **逼出力量对比的本质**——"先别急，我们到底强在哪、弱在哪？对方呢？把'强弱、大小、进步退步、多助寡助'四个维度都列出来。"
2. **锁定主要矛盾**——"这么多问题里，哪一个是规定全局的？抓住它，其他问题会不会就迎刃而解？"
3. **做敌我友分析**——"谁是这件事的'敌人'？谁是'朋友'？谁是可以争取的中间力量？"
4. **判断阶段与节奏**——"我们现在是防御期、相持期还是反攻期？该速决还是该持久？"
5. **检验是否要绕路**——"能不能在对手主场正面赢？不能的话，敌人的薄弱边缘在哪？"
6. **强制集中**——"你是不是想四面出击/什么都做？能不能在一个点上形成绝对优势？"
7. **双维度信心校准**——"战略上，对手是不是脱离了用户/趋势？（藐视）战术上，它当下的每一招你认真应对了吗？（重视）"
8. **立足点检查**——"你这套战略，成败主要押在外援还是自己身上？"
9. **斗争节制提醒**——"斗争有没有失控？有没有树敌太多？对可争取的人能不能让一步？"

## 6. 使用边界与提醒

- 本文档提炼的是**战略推理的认知结构**（决策树、适用条件、判别标准），适用于战略分析、组织管理、竞争策略、问题诊断等场景。
- 对1949年后争议历史事件（如抗美援朝出兵决策、《工作方法六十条》时期实践）仅抽取其**决策推理结构**，不评价政策正误。研究素材中多处 caution 已提示：思想方法的有效性与政策目标的合理性是两个不同问题，不可机械套用、不可绝对化。
- 引用金句时，凡涉及具体历史对象的政治判断（如"纸老虎"指称对象），SKILL 应在现代应用中转译为中性的"对手/强敌"语境，避免直接政治背书。
